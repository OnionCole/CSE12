Lab Description:
Using MIPS and Mars, write a program which takes a natural number from the user, rejecting any unnacceptable input with an error message and a repeated prompt. Do this prompt with syscall 4.
Given the input N, print a "triangle" in the console like so (if N == 3):
		1
	2	*	3
4	*	5	*	6
Then print another newline and syscall 10 to exit.
